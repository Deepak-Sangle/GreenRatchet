# Product Overview

## Purpose

GreenRatchet is a cloud sustainability monitoring platform. It connects to cloud providers (AWS, GCP, Azure) to track environmental metrics and provide actionable insights for optimizing cloud infrastructure sustainability.

## Target Users

- Cloud Operations Teams managing cloud resources
- Sustainability Officers tracking environmental KPIs
- ESG Compliance Teams for environmental reporting

## Key Features

- Multi-cloud integration (AWS, GCP, Azure)
- Real-time environmental tracking (carbon footprint, energy, water)
- KPI dashboard with analytics and trends
- Electricity Maps integration for grid data
- Complete audit trail

## KPI Types

- CO2 Emissions (total and per region)
- GHG Intensity (per employee, per revenue)
- Energy Consumption (kWh)
- Water Withdrawal (liters)
- AI Compute Hours
- Renewable Energy %
- Carbon-Free Energy %
- Low-Carbon Region %
- Electricity Mix Breakdown
- Water-Stressed Region Analysis
